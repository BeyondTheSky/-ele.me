4.1 Octopus移动端SDK使用手册

1.下载SDK工程：是个Android Library工程，需要引入工程

  • 要接入Octopus服务，您需要下载以下一个Android Library工程并将其引入您的Android开发工程
    • Octopus移动端SDK工程

2.在工程清单AndroidManifest中进行配置

1.您需要在AndroidManifest.xml中给予必要的权限

    <uses-permission android:name="android.permission.INTERNET" />
    <uses-permission android:name="android.permission.READ_PHONE_STATE" />
    <uses-permission android:name="android.permission.ACCESS_NETWORK_STATE" />
    <uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />
    <uses-permission android:name="android.permission.INTERACT_ACROSS_USERS_FULL" /> 
    <uses-permission android:name="android.permission.CALL_PHONE" />

2.您还需要在AndroidManifest.xml中对Application进行设置

    <application>标签中必须加上 android:name="com.activeandroid.app.Application"

3.同样还需要在AndroidManifest.xml中对Activity进行设置 

    需再配置清单中引入如下代码：

3.在工程代码中进行配置

1.OctopusSDK在移动端封装了一个类供开发者进行数据获取和服务窗开启

类名：OctopusSDK
实例获取：OctopusSDK.genInstance()

后续的所有Octopus操作及函数都在该类下

2.在使用Octopus所有服务之前，您需要先对Octopus的环境进行初始化（建议在用户信息获取后即刻进行）

/**
 * 初始化方法 step1
 * @param context 上下文实例
 * @param uid 当前用户在您体系中的id
 * @param channelid 渠道商id
 * @param appkey 渠道商注册时的appkey
 * @param callback 初始化的回调函数，包括onSuccess和onError
 */
public void initSDK(Context context,String uid,String channelid,String appkey,InitResultCallback callback)


调用实例：

OctopusSDK.getInstance().initSDK(MainActivity.this, uid, channelid,
            appkey, new InitResultCallback() {

        // 初始化成功后回调函数
        @Override
        public void onSuccess() 
            // TODO 页面跳转等
        }

        // 初始化失败后回调函数
        @Override
        public void onError(String msg) {
            // TODO 
        }
});

3.初始化完成后Octopus会回调initSDK函数中所传入的回调接口：InitResultCallback

public interface InitResultCallback {
    /**
     * 初始化成功后回调onSuccess函数
     */
    public void onSuccess();

    /**
     * 失败后回调onError函数
     * @param msg 错误信息
     */
    public void onError(String msg);
}

4.初始化完成后，您可以选择对Octopus的风格进行定制：可以通过color、R文件id、drawable、dp单位的height数值设定Octopus的标题栏背景和高度，还可通过color设定Octopus标题栏的图标和文字颜色

/**
 * 1.设置octopus的title风格
 * @param rid 通过R文件id设置
 */
public void setOctopusTitleBackgroundResource(int rid)

调用实例：
OctopusSDK.getInstance().setOctopusTitleBackgroundResource(R.drawable.ic_launcher);

/**
 * 2.设置octopus的title风格

 */

/**

/**

调用实例：
OctopusSDK.getInstance().clearOctopusTitleStyle();

5.在设置完Octopus服务窗风格后，您可以开启服务窗

/**
 * 启动交互服务(根据style不同启动对话框形式、或主页形式)
 * @param fromActivity activity上下文
 * @param serviceid 服务在Octopus平台中的id
 * @param style 启动类型：分为对话框模式、微主页模式
 */
public void startDialog(Activity fromActivity,String serviceid,String style)

调用实例：
// 1 进入主页形式服务窗
OctopusSDK.getInstance().startDialog(MainActivity.this,serviceid, OctopusSDK.SERVICE_STYLE_WEB);

// 2 进入对话框形式服务窗
OctopusSDK.getInstance().startDialog(MainActivity.this,serviceid,OctopusSDK.SERVICE_STYLE_DIALOG);

6.除了开启服务窗，您还可以通过OctopusSDK获取当前渠道下的一些数据信息，需要注意的是：以下的方法都需要在线程中开启(建议使用Asynctask等)

/**
 * data1.1 获取当前渠道下所有服务列表数据
 * @param pageno 当前页码(从0开始)
 * @param pagesize 每页条数
 * @return 服务列表结构化数据
 */
public S2CServiceListInfo getServicelist(String pageno,String pagesize)

调用实例：
S2CServiceListInfo services = OctopusSDK.getInstance().getServicelist("0","100");

-------------------------------------------------------------------------

/**
 * data1.2 获取所有类型列表(包括子类型)
 * @return 类型列表结构化数据
 */
public ServiceTypeInfo getTypelist()

调用实例：
ServiceTypeInfo types = OctopusSDK.getInstance().getTypelist();

-------------------------------------------------------------------------

/**
 * data1.3 获取指定类型下当前渠道的服务列表
 * @param tid 类型id
 * @return 服务列表结构化数据
 */
public S2CServiceListInfo getServicelistByType(String tid)

调用实例：
S2CServiceListInfo servicesByType 
    = OctopusSDK.getInstance()
        .getServicelistByType("c95162f5-f82c-431c-8774-947e55dbe33d");

-------------------------------------------------------------------------

/**
 * data1.4 获取当前渠道下关键字查询返回的服务列表
 * @param keyword 搜索使用的关键字
 * @return 服务列表结构化数据
 */
public S2CServiceListInfo getServicelistByKeyword(String keyword)

调用实例：
S2CServiceListInfo servicesByKey = OctopusSDK
    .getInstance().getServicelistByKeyword("石");
