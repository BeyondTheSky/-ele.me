Octopus数据接口OpenApi使用说明

3.4.供“渠道商”调用数据接口

3.4.1 通用header参数

  • 通用头参数，服务端可以通过以下参数确定使用者的身份及权限

    序号 参数名 类型 释义 是否必须
    1 channelid String 渠道商的id 是
    2 appkey String 渠道商的appkey 是

3.4.2 获取远端服务列表

  • 服务地址

    http://<ip>:<port>/channel/client/servicelist/{servicetype}/{pageno}/{pagesize}

  • 请求方式

    GET

  • 输入参数

    序号 参数名 类型 释义 是否必须
    1 servicetype String 获取的服务列表类型(全部、某类型)，全部类型则此参数为all 是
    2 pageno String 需请求的当前页(从0开始) 否
    3 pagesize String 每页的条数 否

  • 输出

    {
    "services": [
        {
            "id": 1,
            "sId":  "这里是服务id",
            "dId": "这里是开发者id",
            "sname": "这里是服务名",
            "surl": "这里是服务的url入口地址",
            "simg": "这里是服务图标url",
            "desripton": "这里是服务描述",
            "token": "这里是注册服务的时remote提供的token",
            "createTime": "1\n407291402654",
            "appId": "这里是appid",
            "secrityId": "这里是secrityId",
            "channalid": "渠道商id",
            "typeid": "分类id",
            "keyword": "服务关键字"

        }
    ],
    "pageInfo": {
        "pageTotalCount": "这里是总页数",
        "pageNo": "这里是当前页数，从0开始",
        "pageSize": "这里是当前分页每页信息条数"
    },
    "code": "601" //601是作为服务列表服务返回代码,失败则为404
    "msg": "错误消息"
}

3.4.3 获取服务类型列表

  • 服务地址

    http://<ip>:<port>/channel/client/typelist/{tid}

  • 请求方式

    GET

  • 输入参数

    序号 参数名 类型 释义 是否必须
    1 tid String 类型id（最大类用all） 否

  • 输出

    {
    "types": [
        {
            "id": 1,
            "tid": “这里是类型id”,
            "parentid": "这里分类的父节点id",
            "tname": "这里是分类名称",
            "des": "这里是类型描述",
            "timg": "这里是分类图标",
            "sub_type": [
                {
                    "id": 32768,
                    "tid": "子类型id",
                    "parentid": "子类型父节点id",
                    "tname": "分类名称",
                    "des": "分类描述",
                    "timg": "这里是子类型分类图标"
                },
                ……
            ]
        },
        ……
    ],
    "code": "200",
    "msg": "错误消息"
}

    http://<ip>:<port>/channel/client/servicedetailbyappkey/{appkey}/{channelid}/{sid}
